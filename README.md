# 签到系统 v2.0

多功能货币系统，包含签到、命运、抽卡（二十四节气卡牌）、银行存款四大模块。

## 指令列表

### 签到
| 指令 | 说明 |
|------|------|
| `签到` / `打卡` / `每日签到` | 每日签到，获得随机货币+桃花值 |

### 货币
| 指令 | 说明 |
|------|------|
| `余额` | 查看货币余额、抽卡次数、集齐套数 |

### 命运
| 指令 | 说明 |
|------|------|
| `命运 <金额>` | 命运赌博，赢则获得等额货币，输则扣除全部（100%） |

### 抽卡
| 指令 | 说明 |
|------|------|
| `单抽` | 消耗1次抽卡机会，自动用200货币兑换 |
| `十连` | 消耗10次抽卡机会 |

**抽卡说明：**
- 25%概率获得二十四节气卡牌，75%概率进入奖池获得货币
- 奖池分为三档（三等40%、二等30%、一等5%），每档5级奖励
- 24节气卡牌收集超过70%后重复概率增加
- 连续10次重复出卡后下次必出新卡
- 集齐一套24节气后套数+1，所有卡牌-1

### 节气
| 指令 | 说明 |
|------|------|
| `节气` / `我的卡牌` | 按春夏秋冬四行展示卡牌数量 |

### 银行
| 指令 | 说明 |
|------|------|
| `存款 <数量>` | 存款，按当前总存款确定利率 |
| `取款 <序号>` | 申请取款（第一次确认） |
| `确认取出` | 确认取款（第二次确认） |
| `利率` / `存款利率` | 查看存款利率 |
| `存款查询` / `我的存款` | 查看存款详情 |

**利率规则：**
- 总存款 ≤ 5w：日利率 10%
- 5w < 总存款 ≤ 10w：日利率 5%
- 总存款 > 10w：日利率 2%
- 每笔存款按存入时总存款锁定利率
- 利息按日自动结算到账户

## 配置项

在插件设置中可配置：
- `admin_qq`：管理员QQ号
- `currency_name`：货币名称
- `gacha_enabled`：是否启用抽卡系统（true/false）
- `gacha_cost`：单抽消耗货币（默认200）
- `bank_enabled`：是否启用银行系统（true/false）
- `set_bonus_interest`：每集齐一套的收益加成（默认0.05=5%）
